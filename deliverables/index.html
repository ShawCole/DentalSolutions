  1. Before/After Transformation Ads — Real patient photos (even phone-shot) consistently outperform polished alternatives. Authenticity = credibility. Dr. Carolina's existing B/A gallery is ideal source material.

Batch Resize Commands (ImageMagick)

Run these to generate Meta-ready versions. Creates a meta-ready/ folder with both formats.

# Create output dirs
mkdir -p meta-ready/square meta-ready/story

# 1:1 Square (1080x1080) from 1024x1024 PNGs
for f in hero.png concierge.png patient.png result1.png team.png; do
  convert "redesign/$f" -resize 1080x1080 "meta-ready/square/$f"
done

# 1:1 Square from 4:3 photos (center crop)
for f in "PHOTO-2025-12-17-21-07-18 4.jpg" "PHOTO-2025-12-17-21-07-18 6.jpg" \
         "PHOTO-2025-12-17-21-07-18.jpg" room1.jpg room2.jpg; do
  convert "redesign/$f" -gravity center -crop 960x960+0+0 \
    -resize 1080x1080 "meta-ready/square/$(basename "$f")"
done

# 9:16 Story (1080x1920) from existing 9:16 thumbnails
for f in david_3s.jpg david_5s.jpg david_8s.jpg sarah_3s.jpg sarah_5s.jpg sarah_8s.jpg; do
  cp "redesign/testimonials/thumbs/$f" "meta-ready/story/$f"
done

# 9:16 from 720x1280 (upscale)
for f in anna_3s.jpg anna_5s.jpg anna_8s.jpg; do
  convert "redesign/testimonials/thumbs/$f" -resize 1080x1920 "meta-ready/story/$f"
done
